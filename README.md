## Installation and Usage

Clone the project

```bash
  git clone https://github.com/deeppatel-dev/battleship
```

Go to the project directory

```bash
  cd battleship
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
